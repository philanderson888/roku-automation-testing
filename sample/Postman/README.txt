WebDriver Postman collection

1.) Side load Roku Recommends sample channel.

2.) WebDriver_endpoints file should be imported as Postman collection.

3.) Open "create session" request and populate "ip" field in request body with your Roku Device IP adress.

4.) Start instance of Roku webDriver.

5.) Execute "create session" request to create new session.

6.) Exucute all other request.

7.) Execute "delete session" request to remove session.
